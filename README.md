# Path Tracer
![](img/cornell.2018-10-03_03-05-39z.1695samp.png)
This homework involves writing a path tracer (a simplified version of a ray tracer).

## Features Completed
* First iteration caching
* Material ID sorting
* Stream compaction
* Reflection shading
* Diffuse shading
* Refraction shading
* Depth of Field
